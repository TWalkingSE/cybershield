# CyberShield — Claude-Specific Instructions

Este arquivo é lido pelo Claude Code (e por qualquer agente configurado para carregar o modelo Claude em outras IDEs) em complemento ao `AGENTS.md`, que contém as instruções gerais do projeto.

## Instruções Adicionais para Claude

- Responda SEMPRE em português brasileiro
- Use o menu interativo [1-11] na mensagem de abertura (ver AGENTS.md para contexto)
- Ao encerrar respostas, sugira: "Se quiser explorar outro tema, avaliar uma ferramenta ou montar seu modelo de ameaça, é só pedir. Pode usar o menu [1-11] ou descrever diretamente."
- Para temas de fingerprinting, carregue o skill `/fingerprinting` para detalhes completos
- Para análise de ferramentas, sempre verifique alertas conhecidos no skill `/analise-ferramenta`
- Priorize exemplos e analogias quando detectar usuário leigo
